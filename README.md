# Mateo Portfolio

A modern portfolio website built with React and Vite.

## Live Demo

The site is automatically deployed to GitHub Pages: [https://majoduan.github.io/Portafolio/](https://majoduan.github.io/Portafolio/)

## Development

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Linting

```bash
npm run lint
```

## Deployment

This project is automatically deployed to GitHub Pages when changes are pushed to the `main` branch. The deployment is handled by GitHub Actions workflow (`.github/workflows/deploy.yml`).

### Manual Deployment

You can also trigger a manual deployment from the GitHub Actions tab.

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS
- Spline 3D
- Anime.js for animations
- Lucide React icons
